# Frontend-MJ

Frontend-MJ é um projeto front-end desenvolvido em React + Vite como teste técnico. Ele apresenta uma interface modular e estilizada com SCSS, organizada para facilitar manutenção e escalabilidade.

- React 18
- Vite

- ESLint
